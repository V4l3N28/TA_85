<<Busque software open source que les pueda ser de utilidad para el desarrolo de la aplicación. Describa sus principales características.>>

<VisualStudioCode>:
  -IntelliSense (auto-completar funciones, controlar nuestras variables, ver las definiciones de funciones en distintos lenguajes)
  - Depurador de código el cual ayuda a reconocer problemas en tiempo real
  - Se integra en Git
  - Compatible con múltiples lenguajes de programación
  
<SublimeText>:
  - Función de selección de varias líneas y a la vez se puede escribir en ellas
  - Función <<Find and Replace>> que permite seleccionar y reemplazar rápidamente en cualquier contenido del documento
  - Modo <<sin distracciones>> que permite escribir sin ningún otro elemento que pueda distraernos
  - Paleta de comandos de acceso rápido a una completa lista de funciónes
<Replit>:
  - Colaboración en tiempo real
  - Chat en vivo en el espacio de trabajo
  - IDE en la nube
<Git>: 
  - No depende de un repositorio central
  - Se puede mantener un historial completo de versiones
  - Sistema de trabajo con ramas
<HeroKu>:
  - Es un código que siempre corre dentro de un dyno el cual garantiza la escalabilidad en caso de que una aplicación se convierta en variables
  - Los Dynos pueden ser de tres tipos: web (desarrollo de la aplicación web), worker (ejecuta la base de datos) o cron (emplea conexiones Secure Shell).
  - Compatible con múltiples lenguajes de programación 
<SQLite>:
  - Base de datos de hastta 2 Terabytes
  - Gestión de Base de Datos transaccional, autocontenido, sin servidor ni configuración.
  - Software libre cuyo código fuente es de dominio público y lilcencia GPL
  - Es posible trabajar con Bases de Datos virtuales en memoria, sin archivo físico.
<XAMPP>:
  - Panel de control que sirve para iniciar y atajar cada módulo
  - Domina herramientas como Apache, MYSQL, PHP y Perl
<MySQL>:
  - Permite escojer múltiples motores de almacenamiento para cada tabla
  - Presenta un amplio subconjunto del lenguaje SQL
  - Disponible en casi todas las plataformas o sistemas
  - Búsqueda e indexación de campos de texto
  - Entorno con seguridad y encryptación 
  - Software con Licencia GPL

<<Identifique librerías disponibles en Python que les puedan ser de utilidad para el desarollo de la aplicación. Describa sus principales características.>>

<Matplotlib>:
  - Librería especializada en la greación de gráficos en 2D como lo son los Diagramas (barras, sectores, caja y bigotes, violín, dispersión o puntos, líneas, áreas, contorno) Mapas de color e Histogramas
  - Los gráficos son personalizables y puede cambiar el aspecto de casi todos sus elementos
  - Es posible dibujar varios gráficos en distintos ejes en una misma figura organizados en forma de tabla
  - Integración con pandas.
<Plotly>: 
  - Gráficos interactivos
  - Gráficos almacenados en formato de data JSON para que puedan ser leídos por otros lenguajes de programación

<charts.js>:
  - Es responsivo
  - Modular (Funciones separadas)
  - Open source
  - La documentación de chart.js  está bien organizada y provee información detallada de cada una de las características.

<django>: 
  - Sistema de autentificación de usuarios 
  - Manejo de versiones
  - Trabajo bajo un patrón MVC (Modelo Vista Controlador)
  - Amplia variedad de paquetes de librerías
  - Panel de administración para bases de datos
<flask>:
  - Adaptación a cualquier proyecto por medio de extensiones específicas para el mismo
  - Servidor propio para pruebas
  - Se integra con otras herramientas para incrementar sus funciones 
<bootstrap 3 o 4>:
  - Permite utilizar muchos elementos web
  - Diseño adaptable
  - Grid system
<Tailwind CSS>: 
  - Permite escribir los estilos por medio de clases que se incluyen dentro del código HTML
  - Herramienta que se apoya en PostCSS para la generación del código CSS

<SQL alchemy>:
  - ORM industrial
  - Base de datos de introspección y generación
  - Sistema de construcción Core SQL y una capa de interacción DBAPI

<Node.JS>:
  - Asíncrono y controlado por eventos
  - Sin almacenamiento en búfer
  - Procesos de un solo hilo con bucle de eventos
  - Licencia basada en código abierto MIT
<React.JS>:
  - Composición de componentes
  - Flujo de datos unidireccional
  - Ideal para aplicaciones de alta demanda.
  - Permite el desarrollo de aplicaciones móviles.
<Vue.JS>:
  - Licencia MIT
  - Optimizado
  - Todo el código de un componente se encuentra en un único fichero
  - Comunidad activa
  - Funcionalidades intuitivas y modernas
<HighCharts>:
  - Soporte multiáctil 
  - Gratuito
  - Dinámico
  - información sobre herramientas configurable
  - Rotación de texto
  - Ejes múltiples
  - Soporte DataTime
